> 所有提示词已针对即梦优化，直接复制粘贴使用

---

## 使用说明

1. 打开即梦，选择「图片生成」
2. 复制下方提示词粘贴到输入框
3. 图片比例选择 3:4（小红书）或 9:16（抖音）
4. 风格选择「写实」或「摄影」
5. 生成后选最满意的一张，后续用「参考图」功能保持风格一致

## 三、即梦使用技巧

### 保持角色一致性

1. **先生成一张满意的羊绅士**
2. 在即梦中找到该图片，点击「用此图生成」或「参考图」
3. 后续所有包含羊绅士的图片都引用这张作为参考
4. 莓莓和啾啾同理，各选一张最满意的作为参考图

### 调整技巧

| 问题 | 解决方法 |
|------|----------|
| 玩偶不够"毛线感" | 在提示词末尾加"清晰可见的钩织纹理，手工毛线质感" |
| 场景太假 | 加"真实物品，真实场景，生活摄影" |
| 光线不对 | 明确写"早晨自然光"或"暖色灯光" |
| 比例不对 | 检查是否选对了 3:4 或 9:16 |
| 角色细节不对 | 加更多细节描述，如"黑色珠子眼睛，圆形身体" |

### 批量生成建议

1. 每个场景生成4张，选最好的1张
2. 先生成「图1封面」，满意后作为风格参考
3. 然后用参考图功能生成其余3张，保持风格统一
4. 三个角色的参考图确定后，后续所有同居素材都可以复用

