## 七、素材生产节奏

### 每周拍摄日（建议固定）

| 时间 | 任务 |
|------|------|
| 周一 | 写/改本周脚本，确认素材清单 |

### 一次性素材准备（第1周前完成）

- [ ] 三个角色定妆照（正面/侧面/背面/特写，每个角色至少5张）
- [ ] 三个角色合照（各种排列方式）
- [ ] 背景布选定并固定（建议买2块替换）
- [ ] 俯拍支架到位

- [ ] 配音声线测试并确定（每个角色试3个声音选最佳）

## 九、数据复盘节奏

| 时间 | 动作 |
|------|------|
| 每条发布后24小时 | 看播放量、点赞、评论数，记录哪条数据好/差 |
| 每周日晚 | 周复盘：本周最佳/最差内容，分析原因 |
| 每月末 | 月度复盘：总播放、涨粉、最佳内容类型、下月调整方向 |

### 关键指标

| 指标 | 抖音 | 小红书 |
|------|------|--------|
| 核心指标 | 完播率 > 点赞率 > 评论率 | 收藏率 > 点赞率 > 评论率 |
| 第1个月目标 | 单条平均播放 500+ | 单篇平均曝光 1000+ |
| 观察信号 | 完播率>30%说明内容留人 | 收藏率高说明内容有长期价值 |
| 调整信号 | 完播率<15%需要优化前3秒 | 曝光高但互动低说明封面/标题需优化 |

## 十、风险预案

| 风险 | 应对 |
|------|------|
| 拍摄效果不好，玩偶摆不出想要的姿势 | 多用"照片+配音"模式，不依赖复杂动作 |
| 某集数据特别差 | 分析原因（标题？封面？时间？），用储备选题替换下一条 |
| 产能跟不上 | 降频：抖音减到2条/周，小红书减到2条/周，保质量不保数量 |
| 观众只对某类内容感兴趣 | 加大该类型占比，减少其他类型，灵活调整 |
| 7集正片用完 | 启用储备选题，或根据粉丝反馈写新脚本 |
