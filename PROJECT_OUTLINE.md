## 一、账号定位

| 维度 | 定义 |
|------|------|
| **一句话定位** | 让每个人都能笑着动起来——最有趣的运动健身入门指南 |
| **公众号名称** | 运动小日常 |
| **内容主轴** | **多样化运动**（居家轻运动、跑步、瑜伽、力量训练、游泳、骑行等），不局限于单一运动类型，根据选题需要灵活覆盖 |
| **核心读者**（前 3 个月） | **25-40 岁想动但还没动起来的上班族**——试过但没坚持、久坐腰酸、想减肥又怕累 |
| **扩展读者**（粉丝量起来后逐步覆盖） | 刚开始运动的新手、想找到乐趣的运动爱好者 |
| **内容调性** | 轻松、有趣、接地气、不端着——像朋友聊天一样聊运动 |
| **小编人设** | **运动的打工人**——和你一样朝九晚六，也想偷懒但也想健康，不装专业教练，只分享自己真正在用的运动方法 |
| **差异化** | 不贩卖焦虑、不秀身材、不堆术语；用趣味降低运动门槛 |
| **更新频率** | 日更（每天 1 篇推文） |
| **转化钩子** | 每篇文章末尾预留行动号召空间：打卡入口、训练计划领取、社群引导，为后续变现铺路 |

## 二、内容支柱（固定栏目）

设置 **5 个固定栏目 + 周末轻内容**，循环覆盖一周，让读者形成阅读预期：

### 1. 动起来的理由（周一）
- **定位**：动机激发 / 认知刷新
- **内容**：打破运动误区、分享有趣的运动科学冷知识、用故事和案例激发行动
- **示例标题**：《你以为跑步伤膝盖？科学家追了 10 万人告诉你真相》《躺平一周后身体发生了什么》

### 2. 运动知识局（周二）
- **定位**：知识型教程 / 运动科普
- **内容**：动作原理、常见误区、身体科普、运动科学入门知识——让读者"知其所以然"
- **示例标题**：《深蹲做错不如不练！3 张图纠正你最常犯的 3 个错》《跑步到底用前脚掌还是后脚跟？答案可能让你意外》

### 3. 今天练什么（周三）
- **定位**：即拿即用的训练方案（强工具属性）
- **内容**：纯训练计划、跟练 GIF/视频、打卡模板、短时高效训练（10-30 分钟）——读者看完就能直接练
- **示例标题**：《下班回家只要 15 分钟，这套训练让你全身都动到》《下雨天不出门？这套居家训练拿走》

### 4. 运动冷知识 / 趣闻（周四）
- **定位**：趣味科普 / 社交货币（打造"值得转发"的爆款潜力篇）
- **内容**：运动历史、名人轶事、数据趣图、反直觉的运动事实
- **示例标题**：《人类为什么能跑赢马？答案出乎意料》《卡路里是怎么被发明出来算食物的》

### 5. 装备选购指南（周五）
- **定位**：消费决策科普
- **前期策略（1-3 个月）**：只做选购方法论和科普（如"跑鞋怎么选"），**不放具体品牌推荐和购买链接**，避免读者过早形成"带货号"认知
- **后期策略（4 个月起）**：逐步引入品牌推荐和 CPS 商品卡
- **示例标题**：《新手选跑鞋只看这 3 点，不花冤枉钱》《瑜伽垫材质大科普：TPE vs 天然橡胶到底差在哪》

### 周末内容（周六/周日）
- **周六**：**运动故事 + 互动**——真实案例（减肥成功、带娃运动、打工人坚持跑步等），用故事做社交货币。启动周没有 UGC 素材，用编辑自己或身边人的故事过渡
- **周日**：轻内容 + 下周预告——面向所有人的独立成篇内容（不依赖"回顾本周"的上下文），如《运动第一周，你只需要记住这 3 件事》

### 每日打卡机制

每篇文章文末设置统一的**打卡口令**或小程序入口：
- 培养读者每日打开习惯
- 为后续社群运营和私域导流做铺垫
- 示例格式：`今日打卡：我已经动了 ___ 分钟 ✅ 评论区记录你的运动`

### 热点结合机制

日更内容不僵化执行固定栏目，**任何一天的选题都可以被当日/近期热点替换或融合**：

**可结合的热点类型**：
- **节气/季节**：入伏适合聊游泳防暑、冬天聊室内训练、开春聊户外跑步
- **体育赛事**：奥运会、世界杯、马拉松赛季、NBA 总决赛等
- **社会话题**：久坐健康危机、打工人过劳、减肥焦虑等泛健康热点
- **娱乐/影视**：运动题材电影上映、明星健身话题
- **平台热搜**：微博/抖音热搜中的运动相关话题

**热点结合原则**：
1. **运动视角切入**：热点只是引子，最终落回运动知识或行动建议
2. **2 小时内决策**：热点保鲜期短，发现热点后快速判断是否结合，当天决定是否替换原定选题
3. **不硬蹭**：热点与运动无关或关联牵强时不强行结合
4. **储备优先**：提前为可预见的热点（赛事、节气）准备备用选题
5. **被替换内容进备选库**：因热点替换的原栏目内容进入备选库，优先在周末加餐或下周同位置补发

**热点选题示例**：
- 奥运会期间 → 《看完奥运想运动？从这 3 个入门最轻松》
- 入伏高温 → 《这么热的天，最适合你的运动是……》
- 某明星健身上热搜 → 《XX 同款训练，普通人能跟吗？》

## 三、日更生产流程

```
热点扫描 → 选题决策 → 初稿 → 审核清单 → 排版配图 → 发布（20:00） → 数据复盘
```

| 环节 | 说明 | 负责 |
|------|------|------|
| **热点扫描** | 每日下午扫描当日热搜/赛事/节气/社会话题，判断是否有可结合的热点 | 数字员工 |
| **选题决策** | 有合适热点 → 替换当日选题（原选题进备选库）；无热点 → 按栏目排期执行 | 人工确认 |
| **初稿撰写** | 根据选题 + 栏目模板生成初稿 | 数字员工 |
| **审核清单** | 按发布前 checklist 逐项检查（见下方） | 人工 / 数字员工 |
| **排版配图** | 公众号排版、封面图、文内插图 | 数字员工辅助 |
| **发布** | 每晚 20:00 发布 | 手动 |
| **数据复盘** | 按数据复盘框架跟踪和分析（见第八章） | 数字员工 |

### 发布前审核清单

每篇文章发布前必须逐项检查：

- [ ] **事实核查**：科学引用有出处、数据准确
- [ ] **调性检查**：是否符合"轻松不端着"的风格，是否像"运动的打工人"在说话
- [ ] **标题测试**：是否有点击欲望，能否在朋友圈引发好奇
- [ ] **敏感词/合规**：运动健康领域避免医疗声明、绝对化用语
- [ ] **CTA 检查**：每篇是否有明确的行动号召（打卡、留言、领取资料等）
- [ ] **配图检查**：封面图和文内图是否到位，风格是否统一
- [ ] **读者价值**：读者看完能获得什么？（知识/方案/动力/乐趣，至少占一个）

## 四、商业变现路径

分阶段推进，内容先行、变现后置：

### 第一阶段：内容积累期（第 1-3 个月）
- **目标**：建立内容库、稳定日更节奏、积累种子用户
- **变现动作**：暂不变现，专注内容质量和更新频率
- **引流动作**：
  - 朋友圈/微信群首发推广
  - 每篇文章底部引导关注 + 打卡
  - 开始沉淀私域（文末引导加个人微信/企微）
- **关键指标**：日更完成率 ≥ 95%、单篇平均阅读量、私域导流人数

### 第二阶段：流量增长期（第 4-6 个月）
- **目标**：粉丝增长、建立信任、启动轻变现验证
- **变现动作**：
  - 文中软植入运动好物推荐（CPS 分佣，从选购方法论过渡到具体推荐）
  - 开通流量主（满足 500 粉丝门槛后）
  - **免费训练计划 PDF** 下载（验证需求、积累用户信任，为付费课程铺路）
  - **低价轻产品试水**：9.9 元"21 天运动打卡计划"、1 元"新手运动指南 PDF"
- **私域动作**：
  - 建立**免费打卡群**，测试活跃度和运营负荷
  - 公众号 → 个人微信/企微 → 社群/朋友圈链路跑通
- **关键指标**：粉丝数、打开率、分享率、私域人数、轻产品转化率

### 第三阶段：商业变现期（第 7 个月起）
- **目标**：多元变现
- **变现动作**：
  - 品牌广告合作（运动品牌、健康食品、运动 APP）
  - 付费课程 / 训练计划（如「8 周跑步入门」「21 天居家健身」）
  - 社群付费（在免费群验证后，开训练指导群）
  - 电商带货（自营或分销运动装备）
- **关键指标**：单篇广告报价、课程销量、社群转化率

## 五、冷启动引流策略

纯靠公众号自然增长太慢，日更内容必须配合外部引流：

### 第一层：微信生态内
- **朋友圈推广**：每篇新文首发朋友圈，前 30 天每天发
- **微信群投放**：运动/健身/上班族相关社群，以"分享有用内容"方式投放
- **互推**：找同量级或稍大的健康/生活类公众号互推
- **公众号菜单/自动回复**：设置关注后自动推送精选文章合集

### 第二层：跨平台导流
- **小红书**：将图文内容改写为小红书笔记，引流至公众号
- **视频号**：将训练方案做成短视频，评论区引导关注公众号
- **知乎**：回答运动健身类问题，文末引导

### 第三层：种子用户
- 首批导入身边对运动感兴趣的朋友/同事
- 在运动社群（跑步群、健身群）中以内容价值吸引首批读者

### 引流节奏
| 阶段 | 重点动作 | 目标 |
|------|----------|------|
| 第 1 个月 | 朋友圈 + 微信群 + 种子用户 | 首批 100 粉丝 |
| 第 2-3 个月 | 跨平台导流（小红书/视频号）+ 互推 | 累计 500 粉丝 |
| 第 4-6 个月 | 持续跨平台 + 社群裂变 + 免费资料包引流 | 累计 3000 粉丝 |

## 八、数据复盘框架

### 每篇必追核心指标
| 指标 | 说明 | 健康基线 |
|------|------|----------|
| **打开率** | 阅读数 / 粉丝数 | ≥ 5% |
| **分享率** | 分享数 / 阅读数 | ≥ 3% |
| **新增关注** | 该篇带来的新关注数 | 正增长 |
| **留言数** | 评论区互动量 | ≥ 5 条 |
| **打卡参与** | 文末打卡口令参与人数 | 逐周增长 |

### 复盘节奏
| 频率 | 内容 | 负责 |
|------|------|------|
| **日复盘** | 记录当日文章核心数据 | 数字员工 |
| **周复盘** | 本周 7 篇数据对比、最佳/最差篇分析、下周选题调整建议 | 数字员工 |
| **月复盘** | 月度增长趋势、粉丝画像变化、变现数据、下月策略调整 | 数字员工 + 人工 |

### 数据驱动调整规则
- 某类选题连续 2 周数据低于基线 → 调整选题方向或标题风格
- 分享率突然升高 → 分析原因，复制成功模式
- 新增关注持续低迷 → 检查 CTA 设计和引流策略
- A/B 测试：每月至少 2 次标题风格或封面风格对比测试

## 十、启动周特别安排

第一周是新读者认识这个号的第一印象，需要额外注意：

1. **周一或周二安排一篇轻量自我介绍**："我们是谁、为什么做这个号、你能在这里获得什么"，降低新读者认知成本，提升关注转化率
