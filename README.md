# 我的钩织运营计划

## 项目概述

钩织玩偶内容创作项目，通过"玩偶剧场"形式打造差异化内容IP，在小红书和抖音双平台运营。

## 核心模式

用钩织玩偶作为角色，通过AI辅助写剧本 + 俯拍摆拍 + 配音字幕，制作短段子故事视频。

## 三个角色

| 角色 | 原型 | 性格关键词 | 角色功能 |
|------|------|-----------|----------|
| 羊绅士 | 翘屁屁小羊（帽子+手杖） | 装绅士、怂、爱面子 | 搞笑担当 |
| 莓莓 | 草莓小人（有短腿） | 嘴毒、冷静、一针见血 | 吐槽担当 |
| 啾啾 | 小山雀（翅膀不能展开） | 话痨、社牛、飞不起来 | 气氛担当 |

## 项目文件夹结构

```
钩织玩偶剧场/
├── 角色设定/          # 角色性格、关系、梗库、配音设定
├── 脚本/              # 每集详细拍摄脚本
├── 素材/
│   ├── 定妆照/        # 三个角色的标准定妆照
│   ├── 拍摄原片/      # 每次拍摄的原始照片/视频
│   └── 配音/          # 配音音频文件
├── 发布内容/
│   ├── 抖音/          # 抖音视频成品和文案
│   └── 小红书/        # 小红书图文和文案
└── 发布计划/          # 内容日历、排期、数据复盘
```
