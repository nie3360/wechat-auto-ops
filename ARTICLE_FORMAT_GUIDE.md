# 公众号文章排版 & 配图插入规范

> 适用对象：内容运营专员（排版执行）、视觉设计师（配图交付）
> 最后更新：2026-06-26

---

## 一、公众号编辑器排版参数

### 1.1 正文排版

| 参数 | 设置值 | 说明 |
|------|--------|------|
| **正文字号** | 16px | 公众号默认推荐，适合手机阅读 |
| **正文行高** | 1.75 倍（28px） | 保证阅读呼吸感 |
| **正文颜色** | `#3F3F3F` | 比纯黑柔和，长时间阅读不刺眼 |
| **正文字体** | 系统默认 | 公众号编辑器不支持自定义字体 |
| **段落间距** | 段后 1 行空白（空一行） | 段落之间明显分隔 |
| **首行缩进** | 不缩进 | 公众号文章惯例，顶格书写 |
| **文字对齐** | 左对齐 | 不使用两端对齐 |

### 1.2 标题排版

| 层级 | 字号 | 字重 | 颜色 | 段前/段后间距 | 使用场景 |
|------|------|------|------|---------------|----------|
| **文章大标题** | 22px | Bold | `#2B2B2B` | 段前 0 / 段后 20px | 文章标题（发布时填入标题栏） |
| **一级小标题** | 20px | Bold | `#2B2B2B` | 段前 24px / 段后 12px | 文章内主要分段标题 |
| **二级小标题** | 18px | Bold | `#FF7A45`（主色） | 段前 16px / 段后 8px | 段落内重点小标题 |
| **三级小标题** | 16px | Bold | `#2B2B2B` | 段前 12px / 段后 6px | 列表项标题 |

### 1.3 强调与特殊文字

| 元素 | 样式 | 示例 |
|------|------|------|
| **正文加粗** | 16px Bold `#2B2B2B` | 核心观点、关键结论 |
| **主色强调** | 16px Bold `#FF7A45` | 金句、核心行动建议（每篇不超过 3-5 处） |
| **引用文字** | 左侧 3px 竖线（`#FF7A45`），背景 `#FFF7F0`，文字 `#666666`，16px | 研究引用、名人语录 |
| **注释/来源** | 14px `#999999` | 数据来源、参考文献标注 |
| **超链接** | `#576B95`（微信默认蓝） | 不使用下划线 |

### 1.4 列表排版

| 类型 | 格式 | 说明 |
|------|------|------|
| **无序列表** | 使用 emoji 作为项目符号（如 ✅ 💡 🏃） | 比默认圆点更有品牌感 |
| **有序列表** | 数字 + 点（1. 2. 3.） | 步骤说明、排名 |
| **列表项间距** | 项间空半行 | 保持呼吸感 |
| **列表缩进** | 左缩进 16px | 与正文区分层级 |

### 1.5 分隔线

| 场景 | 样式 | 说明 |
|------|------|------|
| **章节分隔** | 纯色细线 `#E8E8E8`，1px | 公众号编辑器自带分隔线 |
| **图片前后** | 图片上下各留 1 行空白 | 图片不紧贴文字 |
| **板块过渡** | 可用 emoji 分隔（如 ··· 或 ———） | 轻松调性场景 |

### 1.6 页眉/页脚固定元素

**每篇文章底部固定模块（按顺序）：**

```
---分隔线---

【打卡口令区】
  打卡口令：「xxx」
  引导文案 + 格式模板

---分隔线---

【互动引导区】
  转发引导
  评论引导
  关注引导

---分隔线---

【预告区】（可选）
  明日预告一句话
```

## 二、配图插入规范

### 2.1 配图类型与位置

| 配图类型 | 插入位置 | 与上下文的间距 | 说明 |
|----------|----------|----------------|------|
| **封面图** | 不在文章内，发布时单独设置 | — | 900×383px，在公众号发布界面设置 |
| **开头配图/情绪图** | 文章第一句话之前或第一段之后 | 上 0px / 下 1 行空白 | 破冰用，调节阅读情绪 |
| **章节配图** | 一级小标题之后、正文之前 | 上 1 行空白 / 下 1 行空白 | 配合章节主题的信息图或场景图 |
| **对比图/数据图** | 紧跟相关论述段落之后 | 上 1 行空白 / 下 1 行空白 | 论据可视化，增强说服力 |
| **动作示意图** | 训练方案段落中，每个动作说明之后 | 上 12px / 下 12px | 配合文字说明 |
| **GIF 动图** | 替代或补充动作示意图 | 同动作示意图 | 优先用于动作教学场景 |
| **品牌卡片** | 文章末尾打卡区之前 | 上 1 行空白 / 下 1 行空白 | 号设卡/打卡卡/互动卡/预告卡 |
| **表情包** | 文章开头、段落过渡、文末打卡前 | 上 12px / 下 12px | 每篇最多 1-2 张 |

### 2.2 配图顺序模板（以标准日更文章为例）

```
┌─────────────────────────────────────────┐
│  文章标题                                │
├─────────────────────────────────────────┤
│  开头文字（钩子段落）                      │
│                                         │
│  [表情包] ← 可选，破冰用                  │
│                                         │
│  正文段落...                             │
│                                         │
│  ─── 分隔线 ───                          │
│                                         │
│  ## 一级小标题                            │
│                                         │
│  [章节配图/对比图/数据图]                  │
│                                         │
│  正文段落...                             │
│                                         │
│  ─── 分隔线 ───                          │
│                                         │
│  ## 一级小标题                            │
│                                         │
│  [动作示意图/GIF] ← 训练类文章             │
│                                         │
│  正文段落...                             │
│                                         │
│  ─── 分隔线 ───                          │
│                                         │
│  结尾文字                                │
│                                         │
│  ─── 分隔线 ───                          │
│                                         │
│  [品牌卡片] ← 打卡卡/互动卡/预告卡         │
│                                         │
│  【打卡口令区】                           │
│                                         │
│  【互动引导区】                           │
│                                         │
│  【预告区】← 可选                         │
└─────────────────────────────────────────┘
```

### 2.3 各栏目配图插入差异

| 栏目 | 典型配图数量 | 配图重点 | 特殊说明 |
|------|-------------|----------|----------|
| **动起来的理由**（周一） | 2-3 张 | 数据信息图 + 情绪图 | 数据图需标注来源 |
| **运动知识局**（周二） | 3-4 张 + 1 GIF | 对比图 + 动作分解 + GIF | GIF 用于动作对比 |
| **今天练什么**（周三） | 5-6 张 + 1-2 GIF | 每个动作 1 张 + 打卡卡片 | 动作图统一风格 |
| **运动冷知识**（周四） | 2-3 张 | 趣味信息图 + 场景插画 | 信息图设计感要强 |
| **装备选购指南**（周五） | 2-3 张 | 结构拆解图 + 选购要点卡片 | 卡片方便保存 |
| **运动故事**（周六） | 2-3 张 | 故事时间轴 + 互动引导卡 | 故事感优先 |
| **周末轻读**（周日） | 2-3 张 | 知识精华长图 + 预告卡 | 适合保存转发 |

### 2.4 配图技术要求

| 参数 | 要求 |
|------|------|
| **格式** | PNG（首选）或 JPG（质量 ≥ 85%）；GIF ≤ 2MB |
| **宽度** | 统一 1080px（公众号会自动缩放，但源图保持 1080px） |
| **色彩模式** | sRGB |
| **分辨率** | 72dpi |
| **文件大小** | 单张 ≤ 5MB（公众号上传限制）；GIF ≤ 2MB |
| **命名** | 按 VI_GUIDELINES.md §11.1 命名规范 |

### 2.5 配图交付流程

```
内容运营专员 → 填写配图 brief（在文章 markdown 中以 <!-- --> 注释标注）
     ↓
视觉设计师 → 按 brief + VI 规范出图
     ↓
视觉设计师 → 交付到 articles/ 对应目录，文件命名规范
     ↓
内容运营专员 → 审核配图（风格/内容/尺寸/命名）
     ↓
通过 → 替换文章中的注释标注为实际图片引用
不通过 → 反馈修改意见，视觉设计师修改后重新交付
```

### 2.6 文章 Markdown 中的配图标注格式

在文章初稿中，配图位置用 HTML 注释标注，供视觉设计师识别：

```markdown
<!-- 配图标注格式 -->
<!-- 类型：封面图 / 文内图 / GIF / 品牌卡片 / 表情包 -->
<!-- 尺寸：宽 × 高 px -->
<!-- 内容：详细描述画面内容 -->
<!-- 风格：参考 VI_GUIDELINES.md 对应规范 -->
<!-- 文件命名：按规范命名 -->
```

示例：

```markdown
<!-- 类型：文内图 - 对比图 -->
<!-- 尺寸：1080×600px -->
<!-- 内容：左右对比，左边「以为的累」瘫坐沙发，右边「实际的累」久坐僵硬 -->
<!-- 风格：简笔画，幽默感，参考 VI §8.3 -->
<!-- 文件命名：in_20260630_01_两种累对比图.png -->
```

## 三、各栏目排版模板速查

### 通用结构

```
标题
│
├─ 开头钩子（1-3 段，引发共鸣或好奇）
│   └─ [可选：表情包]
│
├─ 正文主体（2-4 个章节，每章节含小标题）
│   ├─ ## 小标题 1
│   │   ├─ [配图]
│   │   └─ 正文...
│   ├─ ## 小标题 2
│   │   ├─ [配图]
│   │   └─ 正文...
│   └─ ## 小标题 3（如有）
│       └─ 正文...
│
├─ 结尾总结（1-2 段，回扣主题）
│
├─ [品牌卡片]
│
├─ 打卡口令区
├─ 互动引导区
└─ 预告区（可选）
```

### 字数参考

| 栏目 | 建议字数 | 配图数 | 阅读时长 |

| 运动知识局 | 1500-2000 字 | 3-4 张 + GIF | 5-7 分钟 |
| 今天练什么 | 800-1200 字 | 5-6 张 + GIF | 3-5 分钟 |
| 运动冷知识 | 1000-1500 字 | 2-3 张 | 3-5 分钟 |
| 装备选购指南 | 1200-1800 字 | 2-3 张 | 4-6 分钟 |
| 运动故事 | 1000-1500 字 | 2-3 张 | 3-5 分钟 |
| 周末轻读 | 800-1200 字 | 2-3 张 | 3-4 分钟 |
