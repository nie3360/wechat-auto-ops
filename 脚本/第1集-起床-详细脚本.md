# 第1集「起床」完整拍摄脚本

---

## 一、拍摄前准备

### 你需要的东西

| 物品 | 用途 |
|------|------|
| 手机 + 俯拍支架（或用手举着也行） | 拍摄 |
| 纯色背景布（白/米/浅灰） | 铺在桌上，干净画面 |
| 三个玩偶 | 羊绅士、莓莓、啾啾 |
| 一盏台灯或自然光 | 侧面打光，避免阴影太重 |
| 手机剪辑软件（剪映即可） | 后期配音、字幕、拼接 |

### 场景布置

```
        台灯
         ↘
    ┌─────────────┐
    │             │
    │  背景布      │
    │             │
    │  🐑   🍓   🐦  ← 三个玩偶平放，间隔一拳距离
    │             │
    └─────────────┘
         ↑
       手机俯拍
```

- 手机镜头正对桌面，不要歪
- 画面里只留三个玩偶，不要出现杂物
- 光线从侧面打，玩偶有轻微阴影更有立体感

---

## 二、逐镜头拍摄指导

> 拍摄方式：每个镜头摆好位置 → 拍一张照片（或2秒短视频）→ 移动玩偶到下一个位置 → 再拍。最后用剪映把照片/短片段拼起来。

---

### 镜头1：三个睡觉

| 项目 | 内容 |
|------|------|
| 画面 | 三个玩偶平躺，排成一横排，闭眼 |
| 位置 | 画面中间偏下，羊绅士在左，莓莓中间，啾啾在右 |
| 停留 | 2秒 |
| 配音 | 旁白（温柔讲述感）："钩织小屋，早上六点半。" |
| 字幕 | 画面底部居中：「钩织小屋 · 早上 6:30」 |
| 音效 | 安静，加一点闹钟滴答声 |
| BGM | 轻音乐，很轻的钢琴或吉他，音量要低 |

---

### 镜头2：羊绅士"起床"

| 项目 | 内容 |
|------|------|
| 画面 | 羊绅士从平躺变成"坐起来"——身体靠着小杯子或书本撑住，帽子扶正，手杖靠在旁边 |
| 怎么拍 | 把羊绅士摆成靠着杯子的坐姿，帽子戴正，手杖放旁边。拍2秒 |
| 配音 | 羊绅士（装腔作势的中年男声，慢条斯理）："注意，形象。" |
| 字幕 | 羊绅士头上弹出文字气泡：「注意形象。」 |
| 音效 | "叮"一声 |

---

### 镜头3：羊绅士看莓莓

| 项目 | 内容 |
|------|------|
| 画面 | 羊绅士坐姿不变，身体微微转向莓莓方向。莓莓还是平躺不动 |
| 位置 | 画面里能看到羊绅士（左侧）和莓莓（中间），莓莓完全没动 |
| 停留 | 1.5秒 |
| 配音 | 羊绅士（带点嫌弃）："你，不起床的吗？" |
| 字幕 | 羊绅士文字气泡：「你不起床的吗？」 |

---

### 镜头4：莓莓回应

| 项目 | 内容 |
|------|------|
| 画面 | 莓莓还是原位，几乎没动 |
| 停留 | 2秒 |
| 配音 | 莓莓（冷淡女声，语速慢，没有情绪波动）："我起了。只是你看不见我动。" |
| 字幕 | 莓莓文字气泡：「我起了。只是你看不见我动。」 |
| 音效 | 无 |

---

### 镜头5：羊绅士吐槽

| 项目 | 内容 |
|------|------|
| 画面 | 羊绅士微微摇头（身体角度稍微歪一点） |
| 停留 | 1.5秒 |
| 配音 | 羊绅士（更嫌弃）："你从昨晚到现在，一毫米都没挪。" |
| 字幕 | 羊绅士气泡：「你从昨晚到现在一毫米都没挪。」 |

---

### 镜头6：莓莓反击

| 项目 | 内容 |
|------|------|
| 画面 | 莓莓纹丝不动 |
| 停留 | 2秒 |
| 配音 | 莓莓（依然冷淡）："那叫稳定。你学学。" |
| 字幕 | 莓莓气泡：「那叫稳定。你学学。」 |
| 音效 | 轻微"嗯？"音效 |

---

### 镜头7：啾啾冲进来

| 项目 | 内容 |
|------|------|
| 画面 | 啾啾从画面右侧外面突然推进来，停在羊绅士旁边 |
| 怎么拍 | 先拍没有啾啾的画面，再把啾啾放进画面拍。剪映里用"从右侧滑入"动画 |
| 停留 | 1秒 |
| 配音 | 啾啾（话痨少年音，语速快，很兴奋）："起床了起床了！今天干嘛！" |
| 字幕 | 啾啾气泡：「起床了起床了！今天干嘛！」大号字体 |
| 音效 | "嗖"一声 |

---

### 镜头8：啾啾撞羊绅士

| 项目 | 内容 |
|------|------|
| 画面 | 羊绅士被撞歪——身体倾斜，帽子歪到一边，手杖倒了 |
| 怎么拍 | 把羊绅士摆成歪斜姿势，帽子推歪，手杖倒在地上 |
| 停留 | 1.5秒 |
| 配音 | 羊绅士（虚弱、崩溃）："……我的形象。" |
| 字幕 | 羊绅士气泡：「……我的形象。」字体稍微抖动 |
| 音效 | "砰"（碰撞声）+ 玻璃碎裂音效 |

---

### 镜头9：莓莓慢慢挪了一步

| 项目 | 内容 |
|------|------|
| 画面 | 莓莓从原来位置移动了一小段距离（只挪1-2厘米） |
| 停留 | 2秒 |

| 字幕 | 莓莓气泡：「看，我也起床了。」 |
| 音效 | 很慢的"咻"声 |

### 镜头10：啾啾吐槽莓莓

| 项目 | 内容 |
|------|------|
| 画面 | 啾啾转向莓莓方向 |
| 停留 | 1.5秒 |
| 配音 | 啾啾（笑）："你这一小步，走了半小时吧？" |
| 字幕 | 啾啾气泡：「你这一小步走了半小时吧？」 |

---

### 镜头11：莓莓金句

| 项目 | 内容 |
|------|------|
| 画面 | 莓莓正面特写（手机稍微推近，只拍莓莓） |
| 停留 | 2秒 |
| 配音 | 莓莓（认真、一字一顿）："腿短，但每一步，都算数。" |
| 字幕 | 莓莓气泡：「腿短，但每一步都算数。」加粗大字，本集金句 |

### 镜头12：结尾全景

| 项目 | 内容 |
|------|------|
| 画面 | 俯拍全景：羊绅士歪着，莓莓只挪了一点点，啾啾歪着角度 |

| 配音 | 无 |
| 字幕 | 画面中间大字：「钩织小屋的一天，开始了。」 |
| BGM | BGM推大，轻快结尾 |
| 结尾钩子 | 加一行小字：「明天会更乱。关注看下集」 |

## 三、配音指南

| 角色 | 声线 | 语速 | 情绪 | 剪映建议 |
|------|------|------|------|----------|
| 羊绅士 | 装腔作势的中年男声 | 慢，每个字咬得很清楚 | 永远在维持体面 | "成熟男声"或"磁性男声"，语速0.8x |
| 莓莓 | 冷淡女声 | 很慢，没有起伏 | 永远不在乎 | "温柔女声"，语速0.7x，越平越好 |
| 啾啾 | 话痨少年音 | 快，像连珠炮 | 永远很兴奋 | "少年音"或"活泼男声"，语速1.2x |
| 旁白 | 温暖讲述感 | 正常 | 像讲绘本 | "温柔女声"或自己录 |

### 配音节奏
- 每句台词之间留0.5-1秒空白
- 莓莓的台词前后多留0.5秒，强化"冷场"效果
- 啾啾的台词紧接上一句，制造"打断"的感觉

## 四、剪映剪辑步骤

### 1. 导入素材
把所有照片/短片段按镜头顺序导入剪映时间线

### 2. 每张照片时长

| 镜头 | 时长 |
|------|------|
| 1（睡觉） | 2.5秒 |
| 2（羊起床） | 2秒 |
| 3（看莓莓） | 1.5秒 |
| 4（莓莓回应） | 2.5秒 |
| 5（羊吐槽） | 2秒 |
| 6（莓莓反击） | 2.5秒 |
| 7（啾啾冲入） | 1.5秒 |
| 8（撞歪） | 2秒 |
| 9（莓莓挪步） | 2.5秒 |
| 10（啾啾吐槽） | 2秒 |
| 11（金句特写） | 3秒 |
| 12（全景结尾） | 3秒 |
| 总计 | 约27秒 |

### 3. 添加配音
剪映 → 文本 → 朗读 → 输入台词 → 选择对应声线 → 放到对应镜头下方

### 4. 添加字幕气泡
- 羊绅士：方形/绅士风格气泡，白底黑字
- 莓莓：圆形/简约气泡，浅粉底白字
- 啾啾：爆炸/不规则气泡，黄底黑字，字大一号

### 5. 添加音效
剪映 → 音效 → 搜索：
- "叮"（镜头2）
- "嗖"（镜头7）
- "砰"（镜头8）
- "玻璃碎"（镜头8）
- "闹钟滴答"（镜头1）

### 6. 添加BGM
剪映 → 音乐 → 搜索"治愈"、"轻快"、"日常"
- 音量调到20-30%，不要盖过配音
- 镜头11（金句）时BGM音量推到40%

### 7. 导出
- 分辨率：1080x1920（竖屏，抖音比例）
- 帧率：30fps
