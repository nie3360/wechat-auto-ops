## 一、拍摄前准备清单

### 1.1 设备清单

| 设备 | 要求 | 备选方案 |
|------|------|----------|
| 手机 | 1080p以上，支持竖屏拍摄 | 相机+监视器 |
| 俯拍支架 | 稳定、可调节高度 | 三脚架+横臂，或用手举（需稳） |
| 背景布 | 纯色（白/米/浅灰），无褶皱 | 大张卡纸、布料 |
| 台灯 | 可调色温，亮度可调 | 自然光（窗户边） |
| 反光板 | 补光用，避免阴影过重 | 白纸、泡沫板 |
| 小道具 | 小杯子/书本（羊绅士靠垫） | 任何能撑住玩偶的小物件 |

### 1.2 玩偶状态检查

| 角色 | 拍摄前检查项 |
|------|--------------|
| 羊绅士 | 帽子是否端正、手杖是否能立住、身体是否能摆出坐姿 |
| 莓莓 | 是否能平稳放置、短腿是否明显、整体造型是否完整 |
| 啾啾 | 翅膀是否收好、圆脑袋是否饱满、是否能站稳 |

### 1.3 环境准备

- [ ] 桌面清理干净，只留拍摄区域
- [ ] 背景布铺平，无褶皱、无杂物
- [ ] 关闭室内其他光源，只用台灯或自然光
- [ ] 手机充电满电，存储空间充足
- [ ] 手机调成静音，避免拍摄时震动

## 二、灯光方案

### 2.1 主光设置

```
        台灯（主光）
           ↘
    ┌─────────────┐
    │             │
    │  拍摄区域    │
    │             │
    └─────────────┘
           ↑
        手机俯拍
```

- **主光位置**：左上方45度角（台灯）
- **光质**：柔和，避免硬光造成过重阴影
- **色温**：4000-5000K（自然白光），避免偏黄或偏蓝

### 2.2 补光方案

- 如果阴影过重，在右侧放一张白纸作为反光板
- 如果用自然光，选择阴天或窗帘过滤后的光线
- 避免直射阳光，会造成过曝和硬阴影

### 2.3 灯光检查

- [ ] 三个玩偶都能看清，没有过暗区域
- [ ] 阴影方向一致（都从左侧投射）
- [ ] 没有反光或过曝区域
- [ ] 手机预览画面亮度正常

## 三、构图方案

### 3.1 景别定义

| 景别 | 用途 | 画面范围 |
|------|------|----------|
| 全景 | 三个角色同框，展示整体关系 | 三个玩偶完整入画，留有呼吸空间 |
| 中景 | 两个角色互动 | 两个玩偶完整入画 |
| 近景 | 单个角色特写 | 一个玩偶完整入画，占画面2/3 |
| 特写 | 表情/细节强调 | 玩偶头部或关键道具 |

### 3.2 镜头构图图

#### 镜头1：三个睡觉（全景）
```
    ┌─────────────┐
    │             │
    │  🐑  🍓  🐦  ← 平躺，排成一横排
    │             │     间隔一拳距离
    └─────────────┘
```
- 画面居中，三个玩偶在画面中下部
- 留出上方空间，营造"安静早晨"氛围

#### 镜头2：羊绅士起床（中景）
```
    ┌─────────────┐
    │             │
    │  🐑↑        │  ← 羊绅士坐姿，靠着杯子
    │  (杯子)     │     帽子端正，手杖靠旁边
    │             │
    └─────────────┘
```
- 羊绅士在画面左侧，坐姿
- 莓莓、啾啾暂时不在画面内（或虚化在背景）

#### 镜头3：羊绅士看莓莓（中景）
```
    ┌─────────────┐
    │             │
    │  🐑→  🍓    │  ← 羊绅士身体微转向莓莓
    │             │     莓莓平躺不动
    └─────────────┘
```
- 两个角色都在画面内
- 羊绅士在左，莓莓在中，啾啾暂不入画

#### 镜头4-6：羊绅士与莓莓对话（中景/近景交替）
- 保持镜头3的构图，通过角色微调表达情绪
- 镜头5（羊绅士摇头）：羊绅士身体角度歪5-10度
- 镜头6（莓莓反击）：莓莓纹丝不动，强调"稳定"

#### 镜头7：啾啾冲进来（中景）
```
    ┌─────────────┐
    │             │
    │  🐑   🐦→   │  ← 啾啾从右侧入画
    │             │     停在羊绅士旁边
    └─────────────┘
```
- 先拍空画面（只有羊绅士）
- 再把啾啾放进画面，后期用"滑入"动画

#### 镜头8：啾啾撞羊绅士（近景）
```
    ┌─────────────┐
    │             │
    │  🐑↘  🐦    │  ← 羊绅士歪斜，帽子歪，手杖倒
    │  (手杖倒)   │     啾啾"无辜"站在旁边
    └─────────────┘
```
- 重点展示羊绅士的"翻车"状态
- 帽子歪到一边，手杖倒在地上

#### 镜头9-10：莓莓挪步（中景→近景）
- 镜头9：莓莓只移动1-2厘米（几乎看不出）
- 镜头10：啾啾转向莓莓方向，吐槽

#### 镜头11：莓莓金句（特写）
```
    ┌─────────────┐
    │             │
    │     🍓      │  ← 莓莓正面特写
    │   (金句)    │     占画面2/3
    │             │
    └─────────────┘
```
- 手机稍微推近，只拍莓莓
- 这是本集核心镜头，需要稳定、清晰

#### 镜头12：结尾全景（全景）
```
    ┌─────────────┐
    │             │
    │  🐑↘ 🍓  🐦↗ │  ← 三个角色"混乱"状态
    │             │     羊绅士歪，莓莓挪了一点，啾啾歪角度
    └─────────────┘
```
- 回到镜头1的构图，但角色状态完全不同
- 形成对比，营造"一天开始了"的氛围

## 四、道具摆放指导

### 4.1 羊绅士道具

| 道具 | 摆放位置 | 注意事项 |
|------|----------|----------|
| 帽子 | 头顶端正 | 镜头8需要歪到一边 |
| 手杖 | 靠在身体右侧 | 镜头8需要倒在地上 |
| 小杯子/书本 | 靠在背后 | 用于支撑坐姿，不要入画太明显 |

### 4.2 莓莓道具

- 无额外道具，保持造型简洁
- 重点：短腿要明显可见

### 4.3 啾啾道具

- 无额外道具
- 重点：翅膀收好，圆脑袋饱满

### 4.4 背景道具

- 背景布：纯色，无图案
- 可加少量装饰（如小花朵、小叶子），但不要抢戏
- 本集建议：保持简洁，不加额外装饰

## 五、拍摄流程

### 5.1 拍摄顺序优化

建议按以下顺序拍摄，减少重复调整：

1. **先拍全景镜头**（镜头1、镜头12）
   - 三个玩偶初始位置摆好
   - 拍完镜头1后，不要移动，直接拍镜头12的"混乱"版本
   
2. **再拍羊绅士单人镜头**（镜头2、3、5）
   - 调整羊绅士为坐姿
   - 拍完镜头2后，微调角度拍镜头3、5

3. **接着拍羊绅士+莓莓双人镜头**（镜头3、4、6）
   - 保持羊绅士位置，加入莓莓
   - 莓莓全程几乎不动

4. **然后拍啾啾入画镜头**（镜头7、8）
   - 先拍空画面（只有羊绅士）
   - 再把啾啾放入，拍"撞歪"版本

5. **最后拍莓莓特写**（镜头9、10、11）
   - 莓莓移动1-2厘米
   - 手机推近拍特写

### 5.2 每个镜头拍摄步骤

| 步骤 | 动作 |
|------|------|
| 1 | 摆好玩偶位置 |
| 2 | 检查构图（手机预览） |
| 3 | 检查灯光（无过曝/过暗） |
| 4 | 按下拍摄（照片或2秒视频） |
| 5 | 回放检查（是否清晰、构图正确） |
| 6 | 调整到下一镜头 |

### 5.3 拍摄技巧

- **稳定性**：使用支架时，避免触碰桌面；手持时，双手握稳
- **一致性**：每个镜头的色温、亮度保持一致
- **微调**：角色移动幅度要小，体现"定格动画"的细腻感
- **多拍**：每个镜头多拍2-3张，后期选最佳

## 六、常见问题与应对

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 玩偶站不稳 | 桌面不平或玩偶底部不平 | 用双面胶固定，或用隐形支撑物 |
| 阴影过重 | 光源太硬或角度太低 | 加反光板，或调整台灯高度 |
| 画面偏黄 | 色温不对 | 调整台灯色温，或后期白平衡校正 |
| 玩偶模糊 | 对焦不准或手抖 | 手动对焦到玩偶，使用支架或定时拍摄 |
| 背景有杂物 | 拍摄区域没清理干净 | 重新清理，或用背景布完全覆盖 |
| 角色位置偏移 | 拍摄时触碰 | 拍完一个镜头后，不要移动其他角色 |

## 十、导演备注

1. **本集核心**：角色出场，建立人设。重点镜头是镜头11（莓莓金句特写），需要特别用心。

2. **角色状态**：
   - 羊绅士：从"装绅士"到"翻车"，状态变化要明显
   - 莓莓：全程几乎不动，体现"冷静"人设
   - 啾啾：动态感强，体现"话痨"人设

3. **节奏感**：通过镜头时长控制节奏
   - 镜头1-6：节奏较慢，建立氛围
   - 镜头7-8：节奏加快，制造冲突
   - 镜头9-11：节奏放慢，突出金句
   - 镜头12：轻快结尾

4. **预留素材**：多拍一些角色特写、道具细节，后期可能用到。
