# 钩织玩偶剧场 - 角色设定

---

## 羊绅士（翘屁屁小羊）

| 维度 | 设定 |
|------|------|
| 外形 | 翘屁屁、戴帽子、拿手杖 |
| 道具 | 手杖（可拿掉）、帽子（非固定，可换） |
| 性格 | 自认为是绅士，实际又怂又爱装。爱摆pose但经常翻车 |
| 口头禅 | "注意形象。"（每次说完就出事） |
| 弱点 | 怕风（帽子会被吹走）、怕水、恐高 |
| 没手杖时 | 秒变社恐，缩成一团，说话结巴。手杖是他的"勇气道具" |
| 换帽子时 | 性格跟着变。戴草帽=度假模式，戴皇冠=国王模式，不戴帽子=最真实但最没安全感 |
| 角色功能 | 主角/搞笑担当/负责"装"然后被打脸 |

### 羊绅士专属梗库

| 场景 | 梗 |
|------|-----|
| 手杖被拿走 | 立刻缩起来："你……你把我的勇气还给我……" |
| 帽子被换掉 | 照"镜子"："这不是我。这是另一个人。" |
| 没帽子没手杖 | "我今天……谁都不是。" |
| 手杖被当筷子 | "这是绅士的象征！不是用来夹菜的！" |
| 帽子被坐扁 | "我的帽子……它以前是有形状的……" |

## 莓莓（草莓小人）

| 维度 | 设定 |
|------|------|
| 外形 | 草莓造型，有腿但很短，走路摇摇晃晃 |
| 性格 | 看起来甜，实际上嘴最毒。冷静、犀利、一针见血 |
| 口头禅 | "哦。" / "然后呢？"（冷场王） |
| 弱点 | 怕被吃（每次看到水果刀就紧张） |
| 特色 | 腿短走得慢，但嘴不输任何人 |
| 角色功能 | 吐槽担当/负责拆穿羊绅士的伪装 |

### 莓莓专属梗库

| 场景 | 梗 |
|------|-----|
| 被嫌走得慢 | "腿短怎么了？我嘴长。" |
| 被问为什么不跑 | "急什么，我到了你就老了。" |
| 被嘲笑矮 | "我矮，但我看得比你透。" |
| 别人炫耀 | "哦。"（然后一句拆穿） |
| 被夸可爱 | "我不是可爱，我是冷静。" |
| 遇到危险跑不掉 | "跑不了就不跑。面对。"（然后躲到羊绅士后面） |
| 走了一小步 | "腿短，但每一步都算数。" |

## 啾啾（小山雀）

| 维度 | 设定 |
|------|------|
| 外形 | 小山雀，圆脑袋，翅膀收起不可展开 |
| 性格 | 话痨、社牛、什么都想凑热闹。脑子转得快但记性差 |
| 口头禅 | "等等等等——你说啥？" / "我有个想法！"（然后是不靠谱的想法） |
| 弱点 | 记性差，经常忘记自己在干嘛；飞不起来 |
| 翅膀 | 收起不能展开，自称会飞但实际只能跳 |
| 角色功能 | 气氛担当/制造混乱/推动剧情 |

### 啾啾专属梗库

| 场景 | 梗 |
|------|-----|
| 需要飞的时候 | "我在 mentally 飞。你不懂。" |
| 被问翅膀有什么用 | "装饰。你看，多好看。"（然后默默收起来） |
| 看到真鸟飞过 | （沉默三秒）"切。飞得还没我跳得远。" |
| 从高处跳下 | "这不是飞，这是有方向的坠落。风格不一样。" |
| 别人夸它像鸟 | "像？我就是。只是今天翅膀休息。" |
| 被说不会飞 | "我不是不会飞，我是不想飞。" |

## 三者关系

```
羊绅士 ←— 室友 —→ 莓莓
   ↑                  ↑
  "我才是老大"      "呵呵"
   ↑                  ↑
        啾啾（两边捣乱）
```

- **羊绅士和莓莓**：表面互怼，实际最默契。羊绅士装X的时候只有莓莓能一句话拆穿
- **啾啾**：谁都聊得来，但总是搞砸事情。羊绅士嫌它吵，莓莓嫌它笨，但没它又不行
- 三个住在一个"钩织小屋"里（拍摄桌面）

## 配音设定

| 角色 | 声线 | 语速 | 情绪 | 剪映建议 |
|------|------|------|------|----------|
| 羊绅士 | 装腔作势的中年男声 | 慢，每个字咬得很清楚 | 永远在维持体面 | "成熟男声"或"磁性男声"，语速0.8x |
| 莓莓 | 冷淡女声 | 很慢，没有起伏 | 永远不在乎 | "温柔女声"，语速0.7x，越平越好 |
| 啾啾 | 话痨少年音 | 快，像连珠炮 | 永远很兴奋 | "少年音"或"活泼男声"，语速1.2x |
| 旁白 | 温暖讲述感 | 正常 | 像讲绘本 | "温柔女声"或自己录 |
